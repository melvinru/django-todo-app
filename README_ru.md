# Django TODO App

Современное и функциональное приложение для управления задачами (TODO), созданное на Django и Python. Этот проект был разработан в рамках курса [AI Dev Tools Zoomcamp](https://github.com/DataTalksClub/ai-dev-tools-zoomcamp) от [DataTalksClub](https://datatalks.club/).

## Возможности

- **Управление задачами**: Создание, просмотр, редактирование и удаление задач (CRUD).
- **Сроки выполнения**: Установка дедлайнов для задач.
- **Отслеживание статуса**: Отметка задач как выполненных или активных.
- **Современный интерфейс**: Чистый и адаптивный дизайн с использованием кастомного CSS и иконок Phosphor.
- **Компактный режим**: Оптимизированный макет для удобного просмотра списков.

## Установка

Этот проект использует `uv` для управления зависимостями.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/melvinru/django-todo-app.git
    cd django-todo-app
    ```

2.  **Установите зависимости:**
    ```bash
    uv sync
    ```

3.  **Настройте окружение:**
    Создайте файл `.env` в корневой директории (установите `DEBUG=True` и `SECRET_KEY`).

4.  **Примените миграции:**
    ```bash
    uv run python manage.py migrate
    ```

5.  **Запустите сервер разработки:**
    ```bash
    uv run python manage.py runserver
    ```

    Откройте [http://127.0.0.1:8000](http://127.0.0.1:8000) в вашем браузере.
